# Spring boot Project ({{Screen Society}})

## Overview
Movie and TV show community forum application whee users can comment, post and discuss past and upcoming movies and TV shows.

---
<br>
## Features

- REST API
- Creating post
- Commenting on posts
- Reposting
- Liking posts

<br />

## Technologies Used

- Spring Security
- Hibernate
- Lombok
- MySQL
- Spring MVC
- Thymeleaf
- HTML
- CSS
- Swagger
- JUNIT
- Mockito

---

## Installation


Follow these steps to set up and run the application:

1. Clone the Repository
2. Run the Application from your IDE
3. Go to your localhost port
